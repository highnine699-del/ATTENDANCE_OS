Privacy Policy

Attendance OS is a browser extension and web app designed to help LMU students track attendance, calculate safe skips, and plan recovery strategies.

Data Collected

Attendance OS does not collect or transmit personal user data to any remote service. All data is stored locally in the user's browser or extension storage.

  • Attendance data entered by the user
  • Course names, attendance counts, and semester information
  • Optional user settings such as theme preference and pass threshold

Permissions

The extension requests the following permissions solely to support its core functionality:

  • storage — to save attendance data locally in the extension
  • tabs — to interact with open browser tabs when syncing from the LMU portal
  • host permission for https://att3.lmu.edu.ng/* — to inject the content script into the LMU attendance portal for scraping attendance data
  • host permission for https://highnine699-del.github.io/ATTENDANCE_OS/* — to enable the extension/webapp communication bridge

User Data

Attendance OS does not collect, track, or share personally identifiable information. The extension does not transmit data externally.

Third-Party Services

Attendance OS does not load or execute remote code. All code is packaged within the extension.

Contact

If you have questions about this privacy policy, please contact the developer through the project repository or the extension support page.
