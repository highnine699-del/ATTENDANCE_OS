# Extension Icons

This folder needs to contain the following PNG files:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

These icons should represent the Attendance OS brand. You can:

1. Use an online icon generator (e.g., https://www.favicon-generator.org/)
2. Create simple icons with an emoji 📊 or similar
3. Use any image editor to create these three sizes

For now, the extension will work without these icons (Chrome will use a default puzzle piece icon).
